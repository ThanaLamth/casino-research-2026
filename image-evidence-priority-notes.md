# Image Evidence Priority Notes

Use this file to decide which casino review screenshots are worth capturing first.

## Highest-Value Image Types

1. Logged-in cashier, deposit, or withdrawal page
2. Live support screenshot with a real reply
3. Signup or onboarding flow with multiple steps

Why these matter:

- they prove real access
- they support strong product claims
- they create information gain that generic screenshots usually do not

## High-Value Image Types

4. Logged-in dashboard or account area
5. Real-world device photo taken during the review session
6. Licensing, responsible gambling, or regulatory proof
7. Bonus terms screenshot with useful annotation

## Medium-Value Image Types

8. Geo-restriction or access-block screenshot
9. Public page screenshot that proves a specific claim

## Low-Value Image Types

10. Generic homepage or hero banner
11. Community-sourced proof images
12. AI-generated templates or mockups used as evidence

## Working Rule

Before using an image, ask:

- what claim does it prove?
- what new information does it add?
- does it show real access, real friction, or a real review step?

If it only makes the page look fuller, it is probably low-value.
